Summary

The purpose of this project is to analyze the changes in the weather patterns the closer one travels near the equator. In order to conduct this analysis, a random pull of 500 cities from around the world was required through the use of an Open Weather API and a python library.

After the dataset was assembled, Matplotlib was used to plot the changes found in the dataset, which consisted of taking a look at factors like temperature, cloudiness, wind speed, and humidity. This dashboard provides the visualizations and source data that resulted from the analysis.
